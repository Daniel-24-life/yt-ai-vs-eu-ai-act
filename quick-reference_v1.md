## 📋 发布前 5 分钟自检

### 1. 平台标记（1 分钟）
- [ ] YouTube Studio → 勾选 "Altered or synthetic content"
- [ ] 描述区加一句："本视频使用 AI 生成 [配音/画面/音乐]"
- [ ] 评论区置顶："关于 AI 使用说明，详见描述"

### 2. 元数据检查（1 分钟）
- [ ] 用 [Content Credentials Explorer](https://contentcredentials.org/verify) 检查 C2PA
- [ ] 确认 AI 工具是否自动嵌入元数据（Veo/Sora/Runway 等）

### 3. 辖区合规（2 分钟）
- [ ] **欧盟观众**：确认 Art.50 义务（chatbot 披露 / deepfake 标签）
- [ ] **中国观众**：确认《标识办法》（显式 + 隐式双标识）
- [ ] **加州观众**：确认 SB-942（隐式披露字段）

### 4. 发布后（1 分钟）
- [ ] 播放器界面确认标签可见
- [ ] 描述区确认 AI 声明
- [ ] 误标申诉路径：YouTube Studio → 申诉

## 🎯 三层标签速记

```
第一层：平台显式标签（观众看到）
  ↓
第二层：平台隐式水印（机器读取）
  ↓
第三层：制度性标记（法律要求）
```

**三层互不通气，合规责任叠加在创作者头上。**

## ⚠️ 常见误区

| ❌ 误区 | ✅ 实际 |
|---------|---------|
| "AI Act 延期了" | 只延了存量系统水印 4 个月到 12/2 |
| "YouTube 标了就够了" | 制度层标记是独立义务 |
| "水印技术成熟了" | SynthID/C2PA 截图即失效 |
| "我不在欧洲" | Art.50 覆盖向欧盟观众提供服务的提供者 |
| "删除标签就没事" | 中国《标识办法》明确恶意删除也违法 |
