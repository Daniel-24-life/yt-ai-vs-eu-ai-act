## v2 协作需求

v2 版本需要以下协作：

- **法律专家**：首批执法案例分析（Art.50/99 适用）
- **技术专家**：水印技术成熟度评估（SynthID/C2PA/新兴方案）
- **创作者社区**：合规成本调研（不同规模创作者的实际负担）
- **翻译志愿者**：多语言版本（英文/德文/法文）

如有意协作，请通过 GitHub Issues 联系。
